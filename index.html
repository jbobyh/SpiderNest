// ПРАВИЛА //

WASD — движение игрока
ЛКМ — стрелять из дробовика (удержание для автострельбы)
3 жизни — начальное количество. Трать на открытие дверей
ПКМ — открыть перегородку (–1 жизнь) или закрыть (+1 жизнь). Закрыть можно любую клетку кроме той, где находится игрок
Содержимое — видно в соседних закрытых комнатах
Сердечки — собери для +3 жизней (коснись). Разное количество на каждом уровне
Ключи 🔑 — собери все ключи, чтобы разблокировать выход (2-4 в зависимости от уровня)
Враги — в некоторых комнатах. Открытие выпускает их! Уничтожай из дробовика!
Зелёная клетка — выход! Зайди в неё со всеми ключами, чтобы победить
Чёрные клетки — закрыты навсегда, больше не открыть
